Project Overview

ToyCraft Tales is a data intelligence application designed to uncover insights from US Toy Manufacturing Data. This project features comprehensive data visualizations that explore various aspects of the industry, including production trends, regional performance, and market dynamics.

Development

Developed with an emphasis on minimalist, Apple-inspired design principles, ensuring a clean, intuitive, and distraction-free user experience. The application uses Python and Flask on the backend, alongside a modern, responsive frontend layout built with pure, unadulterated HTML5 and CSS3.
